##### 命令     作用
###### log     查看历史提交记录
###### status <-s/--short>  查看本地代码状态
###### clone   从远端克隆别人的代码仓库
###### pull    把最新的代码拉下来，并合并到当前
###### fetch   把最新的代码拉下来
###### pull == fetch + merge
###### push    把本地的提交上传到远端仓库
###### branch  切分支
###### 注意    不同分支的提交记录是并行的，不互通
###### merge   合并后会增加一个合并节点
###### push                  把本地的提交上传到远端仓库
###### branch                切分支
###### checkout -b <分支名>  创建一个分支并转到该分支  没有-b则表示切换分支，同switch
###### switch <分支名>       转到该分支
