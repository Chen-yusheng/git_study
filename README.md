##### 命令
###### add .   向暂存区提交所有文件
###### add filename   向暂存区提交指定的文件
###### commit -m "notes"  向本地仓库提交代码
###### commit -a -m "notes"     越过add直接提交代码
###### clone   从远端克隆别人的代码仓库
###### pull    把最新的代码拉下来，并合并到当前
###### fetch   把最新的代码拉下来
###### pull == fetch + merge
###### push    把本地的提交上传到远端仓库
###### branch  切分支
###### checkout -b <分支名>   创建一个分支并转到该分支  没有-b则表示切换分支，同switch
###### switch <分支名>    转到该分支
